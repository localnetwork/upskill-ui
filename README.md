# Upskill UI

<img src="./public/logo.png" alt="Upskill Logo" width="200" />

**Upskill UI** is an e-learning platform similar to Udemy. Users can browse courses, enroll, and learn online through a modern, responsive interface built with **Next.js**.

## Features

- Browse courses by category
- Enroll in courses and track progress
- Search and filter courses
- Responsive design for desktop and mobile
- User-friendly interface

## Installation

1. **Clone the repository**

```bash

cd upskill-ui
npm install
npm run dev
```
